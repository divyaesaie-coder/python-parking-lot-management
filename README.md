# python-parking-lot-management
A simple Python parking lot management system to park cars, remove cars, and track available parking slots.
# Python Parking Lot Management

A beginner-friendly Python project that simulates a parking lot management system.

## Features
- Park cars
- Remove parked cars
- Check parked vehicles
- Track available parking slots
- Prevent duplicate car entries

## Technologies Used
- Python 3

## How It Works
The program:
1. Allows users to park cars
2. Stores parked car numbers
3. Removes cars from parking
4. Displays parked vehicles
5. Tracks remaining parking slots

## Concepts Used
- Lists
- Loops
- if-else conditions
- User input handling

## Run the Program

```bash
python parking_lot_management.py
